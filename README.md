# Programmation Évolutive - ADOMC

Bienvenue sur le dépôt du projet de **Programmation Évolutive**.

## Projet principal : Flappy Bird AI (NEAT)

Ce dépôt contient le projet complet d'une Intelligence Artificielle apprenant à jouer à Flappy Bird via l'algorithme génétique **NEAT** (NeuroEvolution of Augmenting Topologies).

👉 **[Aller au code source et aux instructions détaillées du projet (Dossier flappy_ai)](./flappy_ai/README.md)**

### Aperçu des fonctionnalités implémentées :
- Moteur physique et rendu pseudo-3D (Pygame)
- Apprentissage non-supervisé par sélection naturelle
- Suivi des statistiques d'évolution en temps réel (TensorBoard)
- Exportation et chargement du meilleur modèle IA généré

*Projet réalisé dans le cadre du cours de Programmation Évolutive.*
